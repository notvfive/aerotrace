# This tool is not complete or near complete, expect bugs!

## Info
AeroTrace is a script designed to monitor network activity around your proximity and fingerprint ~stations and~ access points, alert to new APs ~& stations &~ detect spoofed MAC addresses ~by stations or~ access points that have been logged previously

## Setup
If the database.db doesnt work you can create your own with the steps below
[download sqlite browser](https://sqlitebrowser.org/) to create a "database.db" file and execute the database.db.sql file to create the necessary tables.
